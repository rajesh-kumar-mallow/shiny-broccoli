# Check-in summary with compensation V2

Tampermonkey userscript for [Hubble](https://hubble.mallow-tech.com) that adds a work log summary with month filter, tooltips, and mini-modals on the my check-in data page.

## Install

[Install in Tampermonkey](https://www.tampermonkey.net/script_installation.php#url=https://raw.githubusercontent.com/rajesh-kumar-mallow/shiny-broccoli/main/check-in-summary-with-compensation-v2.user.js)

Or open the [raw script URL](https://raw.githubusercontent.com/rajesh-kumar-mallow/shiny-broccoli/main/check-in-summary-with-compensation-v2.user.js) and let Tampermonkey install it.

## Updates

The script auto-updates from GitHub when Tampermonkey checks for updates (enabled by default in script settings).

To publish a new version:

1. Edit `check-in-summary-with-compensation-v2.user.js`
2. Bump `@version` in the script header (e.g. `2026-06-25`)
3. Commit and push to `main`

Tampermonkey compares the remote `@version` with your installed copy and prompts to update when a newer version is available.
